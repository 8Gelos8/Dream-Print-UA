# Dream Print UA - Сайт магазину сублімаційного друку

Сучасний, інтерактивний веб-сайт для Dream Print UA, який спеціалізується на сублімаційному друку та виробництві персоналізованих подарунків.

##  Характеристики сайту

 **Анімована головна сторінка** - Розкидані картки продукції на робочому столі з красивими анімаціями при наведенні та кліку

 **Галерея продукції** - Розділ для завантажування та управління фотографіями вашої продукції

 **Прайс-лист** - Детальна інформація про ціни на всі послуги з описом

 **Сторінка послуг** - Огляд всіх доступних послуг:
-  Свічки
-  Брелоки
-  Підвіски
-  Вироби з епоксидки
-  Вишивка
-  Гіпсові фігурки
-  Фігурки з пластику

 **Адаптивний дизайн** - Ідеально відображається на мобільних телефонах, планшетах та комп''ютерах

 **Сучасний UI/UX** - Побудовано на Next.js 16, React 19, TypeScript та Tailwind CSS

##  Швидкий старт

### Вимоги
- Node.js 18+ або новіше
- npm, yarn, або pnpm

### Розпочати локальну розробку

```bash
# Перейти в папку проекту
cd dream-print-ua

# Встановити залежності
npm install

# Запустити сервер розробки
npm run dev
```

Сайт буде доступний за адресою: **http://localhost:3000**

### Побудувати для виробництва

```bash
# Збудувати статичний сайт для GitHub Pages
npm run build

# Протестувати виробничу версію локально
npm start
```

##  Розгортання на GitHub Pages

### Крок 1: Підготовка репозиторію GitHub

1. Створіть новий репозиторій на GitHub з назвою `dream-print-ua`
2. Заберіть сайт на GitHub через git:

```bash
git remote add origin https://github.com/yourusername/dream-print-ua.git
git branch -M main
git push -u origin main
```

### Крок 2: Налаштування GitHub Pages

1. Перейдіть до **Settings** вашого репозиторію
2. Оберіть **Pages** в лівому меню
3. Виберіть **GitHub Actions** як джерело для розгортання
4. Натисніть **Save**

### Крок 3: Автоматичне розгортання

При кожному push в гілку `main`:
- GitHub Actions автоматично будує сайт
- Розгортає його на GitHub Pages
- Сайт буде доступний за адресою: `https://yourusername.github.io/dream-print-ua/`

##  Редагування контенту

### Додавання нових продуктів

Редагуйте файл **src/data/products.ts**:

```typescript
export const products = [
  {
    id: 1,
    name: "Назва продукту",
    nameEn: "English Name",
    icon: "",
    description: "Опис продукту"
  },
  // ...
];
```

### Оновлення цін

В тому ж файлі оновіть секцію `pricing`:

```typescript
export const pricing = [
  {
    id: 1,
    name: "Назва послуги",
    minPrice: 100,
    maxPrice: 500,
    unit: "грн/шт",
    description: "Опис ціни"
  },
  // ...
];
```

### Завантаження фото продукції

1. Перейдіть на сторінку **Галерея** (`/gallery`)
2. Натисніть на область завантажування або перетягніть фото
3. Фото будуть збережені в браузері (локальне сховище)

##  Налаштування

### Змінити назву сайту

Оновіть файли:
- `src/components/Navigation.tsx` - логотип в навігаційній панелі
- `next.config.ts` - `basePath` для GitHub Pages

### Змінити кольори

В компонентах використовуються Tailwind класи:
- Фіолетовий: `from-purple-500 to-indigo-600`
- Можна замінити на інші кольори Tailwind

### Додати контактну інформацію

Редагуйте `src/app/about/page.tsx` та оновіть телефон, email та адресу.
